目下载和运行
----

- 拉取项目代码
```bash
git clone
cd
```

- 安装依赖
```
yarn install
```

- 开发模式运行
```
yarn run dev
```

- 编译项目
```
yarn run build
```

- Lints and fixes files
```
yarn run lint
```



其他说明
----
- 生成环境默认不加载 `mock`，更多详情请看 `src/mock/index.js`
